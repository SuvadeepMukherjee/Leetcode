# Loops

- while loops are similar 

  ```python
  n=0
  while n<5:
  	print(n)
  	n+=1
  ```

- For loops 

  - looping from i=0 to i=4 

    ```python
    for(i in range(5)):
    	print(i)
    ```

  - Looping from i=2 to i=5

    ```python
     for(i in range(2,6)):
     	print(i)
    ```

  - Looping from i=5 to i=2 

    ```python
    for(i in range(5,1,-1)):
    	print(i)
    ```

  - Syntax : `start,end,increment/decrement`
