## Assignment Operators

------

- We use the following assignment operators in Python `+=`, `-=`, `*/`,`/=`, `//=`, `**=`, `%=`

- ```python
  prize_money = 625
  
  prize_money+= 5
  print(prize_money)
  #prize_money = 630
  
  prize_money-=5
  print(prize_money) 
  #prize_money = 625
  
  prize_money*=2
  print(prize_money)
  
  # prize_money = 1250
  
  prize_money/=2
  print(prize_money)
  
  # prize_money = 625
  
  prize_money//= 2
  print(prize_money)
  #prize_money = 312
  
  prize_money**=2
  print(prize_money)
  #prize_money = 97344
  
  prize_money%=2
  print(prize_money)
  #prize_money = 0.0
  ```
